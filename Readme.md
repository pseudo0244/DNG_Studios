# DNG Studios 🎵

Welcome to **DNG Studios**, a powerful and elegant **music notation software** built with precision and passion. This project is developed by **pseudo0244** for **Hamsalekha Music School**, owned by the legendary music composer **Dr. Hamsalekha Sir**.

Inspired by **MuseScore**, DNG Studios provides a seamless experience for musicians, composers, and students to create, edit, and share musical compositions effortlessly.

---

## 🛠️ Built With

- **C++** – High-performance core for handling musical notation.
- **CMake** – Cross-platform build system for easy compilation.
- **Qt (GUI)** – Modern and professional UI framework for an elegant user experience.

---

## ✨ Features

✅ **Professional Music Notation** – Create and edit sheet music with an intuitive and minimal UI.  
✅ **Manual Note Entry with Sound Playback** – Enter notes manually and hear them played back in real-time.  
✅ **Score Customization** – Adjust note styles, spacing, and more to suit your composition needs.  
✅ **Import & Export** – Save and open scores in various formats for easy sharing and collaboration.  
✅ **Modern Dark-Themed UI** – A sleek, professional design for an immersive composing experience.  
✅ **Lightweight & Fast** – Built for performance, ensuring smooth and efficient workflows.

## 🎼 About Hamsalekha Music School

Hamsalekha Music School, founded by **Dr. Hamsalekha**, is dedicated to nurturing musical talent and preserving the beauty of classical and contemporary music. This software is developed to enhance the learning and composing experience for students and professionals alike.
